# Formulário de Inscrição

Um formulário simples e responsivo feito em **HTML** e **CSS**, ideal para inscrições em eventos, cursos ou palestras.

## 📄 Descrição

Este projeto consiste em um formulário de inscrição com os seguintes campos:

- Nome completo
- Email
- Telefone
- Data de nascimento
- Tipo de inscrição (Palestrante, Participante ou Voluntário)
- Gênero (Masculino, Feminino, Prefiro não informar)
- Mensagem adicional

O formulário é estilizado com CSS moderno, incluindo cores, tipografia e responsividade.

## 🎨 Tecnologias Utilizadas

- **HTML5** – Estrutura do formulário  
- **CSS3** – Estilização e layout responsivo

## ⚙ Funcionalidades

- Layout limpo e responsivo  
- Campos obrigatórios com validação básica do navegador  
- Botões para enviar ou limpar o formulário
